# 🛠️ Maintix

> AI-Powered Asset Maintenance & Service Management Platform

Maintix is a modern maintenance management platform that gives every physical asset a digital identity using QR codes. It enables organizations to report issues, manage maintenance workflows, track service history, and leverage AI for intelligent issue triage.

Built for the **SMIT Final Hackathon (Track B)**.

---

## ✨ Features

- 🔐 Secure Authentication
- 📦 Asset Management
- 📱 QR Code Generation
- 🌐 Public Asset Pages
- 📝 Issue Reporting
- 🤖 AI Issue Triage (Google Gemini)
- 🔧 Maintenance Workflow
- 📂 Asset History Timeline
- 📊 Dashboard & Analytics
- 📁 Evidence Upload
- 📱 Fully Responsive Design

## 🌟 Core Workflow

```text
Create Asset
      ↓
Generate QR Code
      ↓
Scan QR
      ↓
Public Asset Page
      ↓
Report Issue
      ↓
AI Issue Analysis
      ↓
Assign Technician
      ↓
Maintenance
      ↓
Resolve Issue
      ↓
Asset History Updated
```
